
# PullTalk Extension

Clarify code reviews in 60 seconds — add voice, video, and visual context directly to GitHub and GitLab pull requests.

## Overview
PullTalk is a browser extension designed to make code reviews faster, clearer, and async-first. Engineers can explain complex feedback with **voice, drawings, and visual highlights** in under a minute, reducing confusing text threads and unnecessary meetings.

## Features
- Record your screen and annotate code directly in PRs
- Embed videos natively in GitHub and GitLab comments
- Preserve the "why" behind every code decision

## Installation
- Install from the GitHub Marketplace or Chrome Web Store
- Click the **Record** button in any PR to start leaving video comments
